# 신기술 동향
1. VoIP 보안 위협
- 음섬 패킷을 불법으로 수집 및 조합해 통화 내용을 재생하고 도청하는 위협
2. DDoS
- 여러 대의 장비를 이용하여 대량의 데이터를 한 곳의 서버에 집중적으로 전송, 서버를 마비시킨다.
3. 워터링 홀
- 표적으로 삼은 집단이 주로 방문하는 웹 사이트를 감염시켜, 피해대상이 해당 사이트를 방문하는 것을 기다린다.
4. 백도아
- 시스템 보안이 제거된 비밀통로
5. 크리슈머
- 소비를 통해 욕구를 충족하는 수준을 넘어 자신의 개성을 표현하는 창조적인 소비자.
6. 가상광고
- CG를 이용하여 광고 형태
7. 유비쿼터스
- 라틴어로 '언제나 어디에나 존재한다' 뜻 시간과 장소에 구애받지 않고 언제나 네트워크에 접속할 수 있는 환경
8. MPEG
- 비디오 압축, 오디오 압축, 멀티미디어 다중화 등 멀티미디어 전 분야를 표준화
9. ONS
- RFID를 달아 무선으로 데이터를 송신하는 장치
10. Zero Day Attack
- 보안 취약점이 발견되었을 때 그문제의 존재 자체가 널리 공표되기도 전에 해당 취약점을 악용하여 이루어지는 보안 공격
11. 스머핑
- IP나 ICMP의 특성을 악용하여 엄청난 양의 데이터를 한 사이트에 집중적으로 보냄으로써 네트워크를 마비시킨다.
12. Hacktivism(핵티비즘)
- 자신들의 정치적 목적을 달성하기 위한 수단
13. Tvshing
- 스마트TV에 악성 소프트웨어 설치하여 최고 접근 권한을 획득하는 해킹기법
14. Zombie
- 다른 사용자를 조종하도록 악성코드에 감연된 컴퓨터
15. Splogger
- 스팸과 블로거의 합성어, 다른 사람의 콘텐츠를 무단으로 복사해 자신의 블로그에 게제하는 블로거
16. IP Spoofing
- 다른 시스템과의 신뢰 관계를 속여서 침입하는 크래킹 기술
17. Sniffing
- 네트워크의 중간에서 남의 패킷 정보를 도청하는 해킹 유형
18. SQL injection 공격
- 웹사이트를 무차별적으로 공격하는 과정 데이터베이스 등의 데이터를 조작하는 일련의 공격 방식
19. brute force attack
- 모든 값을 대입하여 공격하는 방법
20. Qshing
- QR코드를 통해 악성 앱의 다운로드를 유도하거나 악성 프로그램을 설치하도록 하는 금융사기 기법
21. 블록체인(Block chain)
- P2P 네트워크를 이용하여 온라인 금융 거래 정보를 온라인 네트워크 참여자의 디지털 장비에 분산 저장하는 기술
22. 암호 화폐
- 온라인 거래에서 안전하게 사용될 수 있도록 해시함수를 이용한 암호화 기술이 적용된 전자 화폐를 의미한다.
23. 가상 화폐
- 지폐나 동전과 같은 실물이 없이 온라인 네트워크 상에서 발행되어 온라인 또는 오프라인에서 사용할 수 있는 디지털 화폐
24. IDS
- 컴퓨터 시스템의 비정상적인 움직임을 실시간으로 탐지하는 시스템
25. MST
- LoopPay사가 개발, 안테나 내장 휴대폰을 마그네틱 결제 단말기에 근접시켜 신용카드 정보를 전송하는 기술
26. Vaporware
- 판매, 배포 계획이 발표되었으나 중단된 소프트웨어
27. O2O
- 온라인 매장에서 오프라인 매장으로 고객을 유치하는 마케팅
28. Data Diet
- 데이터를 압축하고 겹친 정보는 중복을 배제하고 새로운 기준에 따라 나누어 저장하는 작업
29. Beacon
- 전파를 이용하여 주기적으로 프레임 신호를 동기화하고 전송 정보와 데이터를 전달하는 무선 통신 신호 기술
30. Meformer
-  일에 열중하는 사람
31. Informer
- 정보를 알리는 사람
32. ISP
- 인터넷 사용자들에게 관련 서비스를 제공하는 회사를 일컬음.
33. ACN
- AP가 내장된 단말기로 구성된 네트워크
34. Social Curation
- 인터넷에서 다수 사용자가 자신의 취향대로 정보를 가공해 다른 사람과 공유하는 것
35. VBR
- 부호화 할 때 상황에 따라 압축률을 다르게 하는 방법
36. Stereoscope
- 하나의 물체를 다른 각도에서 찍은 두 장의 사진을 동시에 보게하여 입체화 하는 기기
37. TEIN
- 아시아와 유럽을 연결하는 초고속 정보 통신망
38. M2M
- 무선 통신을 이용한 기계 간의 통신
39. Domain Name
- 숫자로 된 IP주소를 사람이 이해하기 쉬운 문자 형태로 표현한 것
40. DRM
- 데이터의 저작권을 보호하기 위한 시스템
- 데이터의 안전한 배포를 활성화하거나 불법 배포 방지
41. CCL
- 저작물 이용 약관
42. HVC
- 차세대 비디오 부호화 표준
43. RSS
- 업데이트가 잦은 사이트의 최신 정보를 확인해 주는 기술
44. Phablet
- 태블릿 기능을 포함한 5인치 이상의 대 화면 스마트폰
45. BYOD
- 개인 소유 단말기를 업무에 사용하는 환경
46. Botnet
- 악성 프로그램에 감여되어 악의적인 의도로 사용될 수 있는 다수의 컴퓨터들이 네트워크로 연결된 상태
47. Carrier Aggregation
- 복수의 주파수 대역을 동시에 사용할 수 있는 대역폭 확장 기술
48. RPO
- IT 시스템 마비시 필요한 데이터를 백업 수단으로 복구할 수 있는 기준점
49. NAS
- 네트워크 상의 다른 기기들에게 파일 기반 데이터 저장 서비스를 제공
50. SON
- 스스로 망을 구성하는 네트워크
51. Miracast
- 와이파이 얼라이언스가 발표한 디스플레이 기술 큰화면으로 모바일 콘텐츠를 전송함
52. EDGE
- 2.5세대 통신망이라 불리는 전송표준 GSM 통신망에 적응 변조 및 패킷 기술 적용
53. HDMI
- 디지털 영상과 음향 신호를 하나의 케이블로 전달하는 멀티미디어 인터페이스
54. Nomophobia
- 스마트폰이 곁에 없을 때 불안감이나 두려움을 느끼는 현상
55. Tajo
- 아파치 하둡 기반의 분산 데이터 웨어하우스 프로젝트. 하둡의 빅데이터를 분석할 때 하둡 분산 파일 시스템으로부터 SQL을 사용하여 파일을 바로 읽어내는 기술
56. 보안
> 기밀성
- 정보와 자원은 인가된 사용자에게만 접근 허용
> 무결성
- 정보는 인가된 사용자만 수정 가능
> 가용성
- 인가 받은 사용자는 언제라도 사용 가능
> 인증
- 정보를 보내오는 사람의 신원을 확인
> 부인 방지
- 데이터를 송수신한 자가 송수신 사실을 부인할 수 없도록 증거를 제공함
> 접근 통제
- 불법적인 접근을 방지하는 과정
57. AR
- 실제로 존재하는 환경에 가상의 사물이나 정보를 합성하여 마치 원래의 환경에 존재하는 사물처럼 보이도록 하는 컴퓨터 그래픽 기법이다
58. VR
- 컴퓨터 등을 사용한 인공적인 기술로 만들어낸 실제와 유사하지만 실제가 아닌 어떤 특정한 환경이나 상황 혹은 그 기술 자체를 의미
59. MR
- 사용자의 현실 세계의 환경과 디지털 제작 콘텐츠를 자연스럽게 혼합하여 두 환경이 공존하며 서로 인터랙션이 가능한 환경입니다
60. RAID
- 여러 하드디스크로 배열을 구성한 디스크에 데이터 블록을 분산 저장한 것
61. XR(혼합현실)
- VR, AR, MR 을 이들의 기술들을 혼합한 것이다.
62. copyright
- 창작자가 저작물을 법적으로 인정하는 권리
63. copyleft
- 지식과 정보가 누구에게나 열려 있어야 한다는 주장 ( 오픈소스 )
